## Steps to Run backend without any problems:

- Move the terminal to backend using:

````
cd backend
````

- Install requirements file using pip:

````
pip install -r requirements.txt
````

- Download the SKELO model and sample assets using downloadModel.py:

````
python downloads.py
````

- Run backend as a python script:

````
python app.py
````
